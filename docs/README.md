# Docs — deep-research-agent

## 仓库代号与定位

- 代号：dp
- 定位：DeepResearch 项目仓库
- 说明：用于深度研究型 Agent 的流程设计、实验验证与迭代。

## 快速开始

1. 先在 docs/PROJECT_GUIDE.md 明确本轮研究目标与边界
2. 跨模块改动前先在 docs/design/ 增加设计说明
3. 研究实验结果同步写入 docs/journal/ 与 docs/CHANGELOG.md

## 本仓库完成标准（DoD）

- 研究链路能稳定复现
- 关键结论可追溯到来源
- 流程改动在 design 或 changelog 有记录
